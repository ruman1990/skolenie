# napis program ktory ziska z inputu cislo a zisti ci je parne alebo nepárne
```python
cislo = int(input("Zadaj cislo: "))
if cislo % 2 == 0:
    print(f"{cislo} je parne cislo.")
else:
    print(f"{cislo} je nepárne cislo.")
```
